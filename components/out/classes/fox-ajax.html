dataType

String

Provided by the fox-actionsheet module.

Defined in fox-ajax/dist/fox-ajax.html:39

Specifies what data to store in the response property, and to deliver as event.response in response events.

One of:

text: uses XHR.responseText.

xml: uses XHR.responseXML.

json: uses XHR.responseText parsed as JSON.

html: uses XHR.responseText.

jsonp: uses jsonp callback.

Default: 'text'
