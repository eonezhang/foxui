Made for Mobile Hybrid App

Onsen UI is a front-end development frame-work for PhoneGap/Cordova hybrid apps. It forcuses on the creation of usable mobile applications. Also, used with Monaca, it is simple to develop backend features and add support

Custom Element-Based

By simply adding a few lines of code, it becomes possible to create the UI components(navigation bars, tab bars, buttons, etc.) necessary for your application.

  • Button
  • Navigation Bar

Other Features

  • Free and Open Source

    Onsen UI is released under the Apache License and development is housed on Github. It is free and open for anyone to use.

  • Selectable Themes

    Provides many different themes, with native-like designs, and you can always build or customize your very own.

  • Screen Transition

    App operations have a native-like user experience (UX) with smooth animations on cross platform PhoneGap/Cordova apps.

  • Bundled with Font Awesome

    Font Awesome is bundled with Onsen UI. This makes it easy to develop and display the exact icons you need.

  • Customizable

    By relying on HTML and CSS, Onsen UI is highly customizable to suit the many needs that come with development and design.

  • GUI Tool Coming Soon

    With drag & drop support, our GUI tool will make all aspects of front-end development much easier. Scheduled for release in early July.
